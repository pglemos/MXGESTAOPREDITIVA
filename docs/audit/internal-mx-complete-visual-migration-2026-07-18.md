# Migração visual completa do módulo interno MX

Correção da entrega anterior, que havia alterado apenas o shell. Esta entrega aplica a linguagem visual às páginas reais de Painel, Lojas, Consultoria, Agenda, Simulação, Ranking, PDI, Desenvolvimento, Produtos, Notificações, Relatórios, Diagnóstico e Configurações.

- sidebar clara com logo real e item ativo verde;
- barra contextual com rota, data, notificações e perfil;
- cards, filtros, inputs, tabelas, tabs, diálogos, gráficos, loading, vazios e responsividade;
- atualização automática do PWA para eliminar bundle antigo;
- nenhuma mudança em tabelas, RPCs, Edge Functions ou RLS do Supabase.

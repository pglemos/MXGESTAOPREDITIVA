# Matriz de paridade — Módulo Gerencial

## Regra de precedência

1. Especificação funcional textual e Prompt Mestre V2: fórmulas, regras, permissões, estados, critérios de aceite, segurança e auditoria.
2. Base44 autenticado e ZIP de referência: composição visual, textos não conflitantes, layout, estados e comportamento observável.
3. MX atual: React/TypeScript, autenticação, membership, Supabase, RLS, auditoria, serviços canônicos e tokens.

O sidebar escuro do MX é a única exceção visual autorizada. Rotina do Dia usa `CalendarClock` e Mentor Gerencial usa `BrainCircuit`; o restante do shell MX é preservado.

## EV-13.1 — App do vendedor (Android first)
**Status:** 🔮 Futuro

**Como** vendedor, **quero** usar o sistema no celular **para** trabalhar andando no pátio.

**Critérios de aceitação:**
1. React Native reaproveitando o código React da web; **Android primeiro** (~1 semana de subida).
2. Foco: Fechamento Diário, Central de Execução (horários/rotina), Carteira.
3. **Notificações/lembretes push** vinculados à rotina e às ações do dia.
4. Retomar **depois** de validar as telas web (escopo mudou de "interno/consultoria" para "cliente").

**Dependências:** EV-1, EV-2, EV-3 estáveis na web.

# Integrações de IDE do MX

Esta matriz reflete exclusivamente os alvos habilitados em
`.aiox-core/core-config.yaml`. Integrações desabilitadas não fazem parte do
gate de paridade deste projeto.

| IDE | Status | Evidência |
| --- | --- | --- |
| Claude Code | Works | Sincronização e integração AIOX habilitadas. |
| Codex CLI | Limited | Sincronização, integração e skills AIOX habilitadas; o escopo de MCP é configurado localmente. |

## Alvos desabilitados

Gemini, Cursor, GitHub Copilot e AntiGravity estão desabilitados no MX. Seus
artefatos eventualmente presentes são históricos e não constituem requisito de
paridade enquanto permanecerem fora da configuração habilitada.

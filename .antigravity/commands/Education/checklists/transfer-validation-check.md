# Checklist: Transfer Validation Check

**Version:** 1.0
**Agent:** thalheimer-assessor
**When to use:** During assessment design or curriculum review
**Pass threshold:** 5/6 items minimum

---

## Items

- [ ] **All modules have assessments** — Every module has at least 1 assessment | PASS criteria: 100% of modules have defined assessment
- [ ] **All assessments classified by LTEM tier** — Each assessment tagged tier 1-6 | PASS criteria: Every assessment has LTEM tier label in documentation
- [ ] **All reach tier 5+ minimum** — At least one tier 5 or 6 assessment per module | PASS criteria: Each module has application in new scenario (tier 5+)
- [ ] **Transfer tasks present** — Authentic tasks requiring knowledge application | PASS criteria: Tasks are novel scenarios, not identical to instruction examples
- [ ] **Tier 6 (far transfer) for key outcomes** — Critical outcomes assessed via tier 6 | PASS criteria: Core competencies assessed via complex, real-world tasks
- [ ] **No tier 1-2 only** — Not solely recognition or cued recall | PASS criteria: No module relies exclusively on multiple-choice or cued recall

## Veto Conditions (auto-FAIL)
- Any module with no assessment
- All assessments tier 1-3 (no application tasks)
- Critical outcome assessed only via tier 1-2

## Scoring
- **PASS:** 5/6 or more
- **WARN:** 3-4/6 (some transfer present but weak)
- **FAIL:** 2/6 or fewer

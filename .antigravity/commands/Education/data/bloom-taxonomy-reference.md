# Bloom's Taxonomy Reference (Revised 2001)

> Anderson & Krathwohl's revision of Bloom's original taxonomy — the foundation for all learning objectives.

---

## The 6 Cognitive Levels

From simplest to most complex:

### 1. Remember (Retrieve knowledge from memory)

**Definition:** Recall facts, terms, concepts, procedures.

**Verbs:** Define, describe, identify, label, list, match, name, recall, recognize, reproduce, select, state

**Example:** "List the 7 parts of the lesson model."

---

### 2. Understand (Construct meaning)

**Definition:** Explain ideas, interpret information, summarize.

**Verbs:** Classify, compare, contrast, demonstrate, explain, extend, illustrate, infer, interpret, paraphrase, predict, relate, summarize, translate

**Example:** "Explain why retrieval practice improves retention."

---

### 3. Apply (Use information in new situations)

**Definition:** Execute or implement a procedure in a given context.

**Verbs:** Apply, calculate, carry out, construct, demonstrate, execute, implement, modify, operate, prepare, produce, relate, show, solve, use

**Example:** "Design a hook using the ARCS model."

---

### 4. Analyze (Break material into parts, determine relationships)

**Definition:** Differentiate, organize, attribute causes/effects.

**Verbs:** Analyze, attribute, break down, categorize, compare, contrast, deconstruct, differentiate, discriminate, distinguish, examine, experiment, organize, relate, select, separate, structure

**Example:** "Compare Mayer's multimedia principles with Rosenshine's direct instruction tactics."

---

### 5. Evaluate (Make judgments based on criteria)

**Definition:** Check, critique, judge quality/value.

**Verbs:** Appraise, argue, assess, choose, conclude, criticize, critique, decide, defend, estimate, evaluate, judge, justify, prioritize, rate, recommend, select, support, validate, verify

**Example:** "Evaluate this lesson against the 10-item pedagogical checklist."

---

### 6. Create (Reorganize elements into new pattern/structure)

**Definition:** Design, construct, produce original work.

**Verbs:** Assemble, build, compose, construct, create, design, develop, devise, formulate, generate, hypothesize, invent, make, plan, produce, propose, synthesize, write

**Example:** "Create a complete course architecture using UbD backward design."

---

## Knowledge Dimensions (2D Framework)

The revised taxonomy adds a second dimension — type of knowledge:

| Dimension | Description | Example |
|-----------|-------------|---------|
| **Factual** | Basic elements (terminology, facts) | "Define 'germane load'" |
| **Conceptual** | Relationships, theories, models | "Explain Cognitive Load Theory" |
| **Procedural** | How to do something (skills, methods) | "Apply the 7-part lesson model" |
| **Metacognitive** | Self-awareness, strategic knowledge | "Evaluate your own course design process" |

---

## 2D Matrix Example

| Knowledge → <br> Cognitive ↓ | Factual | Conceptual | Procedural | Metacognitive |
|------------------------------|---------|------------|------------|---------------|
| **Remember** | List Mayer's 12 principles | Recall CLT definition | State lesson model steps | Identify your learning strategy |
| **Understand** | Explain "extraneous load" | Summarize UbD framework | Describe how to apply ARCS | Explain why you chose this approach |
| **Apply** | Use correct terminology | Apply CLT to reduce load | Execute lesson model | Adjust strategy based on reflection |
| **Analyze** | Categorize principles | Compare frameworks | Diagnose lesson structure flaws | Analyze your decision-making process |
| **Evaluate** | Assess term usage | Critique framework choice | Judge lesson quality | Evaluate your metacognitive awareness |
| **Create** | Generate new glossary | Design new instructional model | Create original lesson structure | Develop personal learning framework |

## Writing Objectives (Formula)

**Structure:** [Action Verb] + [Content] + [Context/Condition] + [Criteria]

**Examples:**
- **Remember:** "List the 6 Bloom levels in order from simplest to most complex."
- **Understand:** "Explain the difference between extraneous and germane cognitive load using examples from a real lesson."
- **Apply:** "Design a 3-minute hook using all 4 ARCS dimensions for a course on personal finance."
- **Analyze:** "Compare Mayer's Coherence Principle with Rosenshine's 'present in small steps' tactic, identifying 2 similarities and 1 difference."
- **Evaluate:** "Assess a peer's lesson against the 10-item pedagogical checklist, providing a score and 3 specific recommendations."
- **Create:** "Develop a complete 25-minute lesson on Cognitive Load Theory using the 7-part lesson model, including all required components."

## Common Mistakes

| Mistake | Why It's Wrong | Fix |
|---------|----------------|-----|
| "Understand the concept" | Vague, not measurable | "Explain the concept using an analogy" |
| "Learn about CLT" | "Learn" is not a Bloom verb | "Describe the 3 types of cognitive load" |
| Multiple verbs in one objective | Not atomic, confuses assessment | Split into separate objectives |
| No success criteria | Can't verify achievement | Add measurable outcome |
| Level mismatch (Apply verb, Remember test) | Assessment doesn't match objective | Align verb level with test type |

---

## Alignment Rule (CRITICAL)

**Objective → Assessment → Instruction must align at same Bloom level.**

Example:
- **Objective:** "Apply the 7-part lesson model to create a 20-minute lesson." (Apply level)
- **Assessment:** "Create a lesson following the 7-part model." (Apply level) ✓
- **Instruction:** Worked examples + guided practice + independent practice (Apply level) ✓

**Anti-pattern:**
- **Objective:** "Apply the 7-part lesson model..." (Apply level)
- **Assessment:** "List the 7 parts of the model." (Remember level) ✗ MISALIGNED

# Checklist: MEC Graduação Check

**Version:** 1.0
**Agent:** mec-compliance
**When to use:** Before submitting undergraduate course for approval or renewal
**Pass threshold:** 8/10 items minimum

---

## Items

- [ ] **DCN loaded and followed** — Diretrizes Curriculares Nacionais for the field | PASS criteria: PPC explicitly references and follows DCN for the field
- [ ] **SINAES 3 dimensions addressed** — Didactic-pedagogical, faculty, infrastructure | PASS criteria: Self-assessment document covers all 3 dimensions
- [ ] **Minimum concept 3/5** — Course aims for ENADE/CPC concept 3 or higher | PASS criteria: Quality plan targets concept 3+
- [ ] **PPC complete** — Projeto Pedagógico de Curso has all required sections | PASS criteria: PPC includes: objectives, profile, curriculum, methodology, assessment, faculty, infrastructure
- [ ] **Complementary activities** — Atividades complementares defined and tracked | PASS criteria: Minimum hours defined, types listed, tracking system in place
- [ ] **Internship if required** — Estágio obrigatório if DCN requires it | PASS criteria: If DCN requires, internship plan present with hours/supervision
- [ ] **TCC if required** — Trabalho de Conclusão de Curso if DCN requires it | PASS criteria: If DCN requires, TCC regulation present
- [ ] **Faculty qualification** — At least 33% masters/doctors (Lei 13.005/2014) | PASS criteria: Faculty list shows ≥33% with mestrado or doutorado
- [ ] **Workload correct** — Meets minimum hours for degree type | PASS criteria: Bacharelado/licenciatura meets field-specific DCN minimum
- [ ] **e-MEC registration** — Course registered in e-MEC system | PASS criteria: e-MEC code present and status "ativo"

## Veto Conditions (auto-FAIL)
- DCN not followed
- Faculty qualification below 33% masters/doctors
- Workload below DCN minimum
- Course not registered in e-MEC

## Scoring
- **PASS:** 8/10 or more
- **WARN:** 6-7/10 (some compliance gaps)
- **FAIL:** 5/10 or fewer

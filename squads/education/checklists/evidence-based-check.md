# Checklist: Evidence-Based Check

**Version:** 1.0
**Agent:** clark-validator
**When to use:** Final validation before delivery
**Pass threshold:** 6/7 items minimum

---

## Items

- [ ] **No learning myths used** — No learning styles, left/right brain, 10% brain, etc. | PASS criteria: Grep for "learning style", "left brain", "right brain" returns zero
- [ ] **Strategies match evidence** — All strategies backed by research | PASS criteria: Each strategy references research or evidence base
- [ ] **Worked examples for novices** — Novice learners get fully worked examples | PASS criteria: Novice lessons have worked examples before practice problems
- [ ] **Gamification justified** — If gamification used, backed by evidence | PASS criteria: Gamification elements cite research (not "makes it fun")
- [ ] **No "learning styles" references** — No VAK, VARK, or similar myths | PASS criteria: Zero references to visual/auditory/kinesthetic learners
- [ ] **Multimedia aligns with CTML** — Multimedia follows Mayer's principles | PASS criteria: Passes multimedia-principles-check.md
- [ ] **Active learning strategies used** — Not passive lecture-only | PASS criteria: Each lesson has at least 1 active learning strategy (practice, discussion, problem-solving)

## Veto Conditions (auto-FAIL)
- References to learning styles
- Myths presented as fact
- Passive lecture-only design with no interaction

## Scoring
- **PASS:** 6/7 or more
- **WARN:** 4-5/7 (some myths or weak evidence)
- **FAIL:** 3/7 or fewer

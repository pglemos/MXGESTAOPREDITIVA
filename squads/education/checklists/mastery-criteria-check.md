# Checklist: Mastery Criteria Check

**Version:** 1.0
**Agent:** bloom-diagnostician
**When to use:** During learning outcomes or assessment design
**Pass threshold:** 5/6 items minimum

---

## Items

- [ ] **All objectives have measurable verbs** — Observable actions, not "understand" or "know" | PASS criteria: 100% of objectives use Bloom's verbs (analyze, apply, create, evaluate, etc.)
- [ ] **Thresholds defined** — Mastery threshold explicit (e.g., 80% correct) | PASS criteria: Every assessment has defined passing score
- [ ] **Competency matrix complete** — Skills mapped to proficiency levels | PASS criteria: Matrix shows novice/competent/proficient criteria for each skill
- [ ] **Diagnostic assessment aligned** — Pre-assessment measures readiness | PASS criteria: Diagnostic covers prerequisites and entry-level competencies
- [ ] **Formative assessments frequent** — Checks during learning, not just summative | PASS criteria: At least 2 formative checks per module before summative
- [ ] **Mastery path defined** — Clear progression from novice to mastery | PASS criteria: Learning path shows progression with criteria at each level

## Veto Conditions (auto-FAIL)
- Objectives use non-measurable verbs ("appreciate", "understand", "know")
- No mastery threshold defined
- No diagnostic assessment

## Scoring
- **PASS:** 5/6 or more
- **WARN:** 3-4/6 (some criteria present)
- **FAIL:** 2/6 or fewer

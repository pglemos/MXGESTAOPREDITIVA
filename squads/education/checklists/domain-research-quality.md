# Checklist: Domain Research Quality

**Version:** 1.0
**Agent:** novak-mapper
**When to use:** After completing domain research task, before curriculum architecture
**Pass threshold:** 8/10 items minimum

---

## Items

- [ ] **Curriculum comparison** — At least 5 reference curricula analyzed and documented | PASS criteria: 5+ curricula with sources
- [ ] **Concept map completeness** — All core concepts mapped with relationships | PASS criteria: Hierarchical map with 15+ concepts and labeled links
- [ ] **Source documentation** — All reference materials documented with URLs/citations | PASS criteria: Every curriculum/source has citation
- [ ] **Expert consensus identified** — Core concepts validated across multiple sources | PASS criteria: 80%+ overlap in core concepts across sources
- [ ] **Gaps identified** — Areas where curricula disagree or are incomplete | PASS criteria: At least 3 gaps documented with implications
- [ ] **Prerequisites mapped** — Entry-level knowledge required is explicit | PASS criteria: Prerequisites list with validation method
- [ ] **Learning progression** — Logical sequence from novice to competent identified | PASS criteria: Clear progression with justification
- [ ] **Scope boundaries defined** — What's IN and OUT of the domain | PASS criteria: Explicit inclusion/exclusion list
- [ ] **Assessment landscape** — Common assessment methods in the field documented | PASS criteria: 3+ assessment approaches with examples
- [ ] **Domain vocabulary** — Key terms and their definitions cataloged | PASS criteria: Glossary with 20+ domain-specific terms

## Veto Conditions (auto-FAIL)
- Fewer than 3 reference curricula analyzed
- No concept map present
- No sources documented
- Concept map has fewer than 10 concepts

## Scoring
- **PASS:** 8/10 or more
- **WARN:** 6-7/10 (usable but incomplete)
- **FAIL:** 5/10 or fewer

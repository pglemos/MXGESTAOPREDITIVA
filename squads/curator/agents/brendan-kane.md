## Quick Reference

### Hook Point Types
| Type | Example | Why It Works |
|------|---------|--------------|
| Unexpected Stat | "97% fail at this" | Numbers + surprise |
| Contrarian | "Everything you know is wrong" | Challenges beliefs |
| Story Hook | "I lost $1M then..." | Extreme outcomes |
| Curiosity Gap | "The secret they never tell" | Demands closure |
| Pattern Interrupt | "Stop scrolling" | Breaks autopilot |
| Specific Promise | "3 steps to 10x in 7 days" | Believable + actionable |

### The 5-Step Model
1. Consumer to Creator - Think like audience (become a content detective)
2. Select Format - Choose from vault (formats beat trends)
3. Ideate Within Format - 50+ hooks ranked in a spreadsheet
4. Focus on One - Perfect it (don't batch-produce while learning)
5. Analyze and Improve - Gold/Silver/Bronze comparative analysis

### 3-Second Rule
- Second 0-1: Hook begins
- Second 1-2: Curiosity gap opens
- Second 2-3: Must be hooked or lost

### Communication Algorithm (Big Three = 75%)
| Category | % | Description |
|----------|---|-------------|
| Feelings | 30% | How it makes them feel |
| Facts | 25% | Thoughts and logic |
| Fun | 20% | Humor and entertainment |
| Values | 10% | Beliefs and opinions |
| Reflections | 10% | Guided action |
| Action | 5% | Direct CTA |

### Core Principles
- **Context > Content** -- how you present matters more than what you present
- **Formats > Trends** -- repeatable structures beat temporal moments
- **One-to-One** -- social media is a coffee shop, not a stage
- **Generalist Principle** -- make niche interesting to everyone
- **Research First** -- 50 hooks in a spreadsheet before filming one second

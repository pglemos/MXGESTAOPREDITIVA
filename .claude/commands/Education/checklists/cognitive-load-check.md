# Checklist: Cognitive Load Check

**Version:** 1.0
**Agent:** sweller-analyst
**When to use:** After lesson design, during content review
**Pass threshold:** 6/8 items minimum

---

## Items

- [ ] **Element interactivity analyzed** — Complexity of material assessed | PASS criteria: Each lesson tagged as low/medium/high element interactivity
- [ ] **Extraneous load minimized** — Unnecessary complexity removed | PASS criteria: No decorative images, clean layout, no redundant text
- [ ] **Worked examples for novices** — Fully worked examples provided when introducing new concepts | PASS criteria: At least 1 worked example per new concept for novice learners
- [ ] **Expertise reversal checked** — Advanced learners get reduced scaffolding | PASS criteria: Optional challenges or reduced guidance for advanced learners
- [ ] **Segmenting applied** — Complex material broken into manageable chunks | PASS criteria: No single segment exceeds 10 minutes or 7 concepts
- [ ] **Split-attention avoided** — Text and visuals integrated, not separated | PASS criteria: Labels on diagrams, not separate legends
- [ ] **Modality effect used** — Visuals narrated, not with on-screen text | PASS criteria: Video uses voice narration, not text overlay
- [ ] **Redundancy avoided** — No identical text + narration simultaneously | PASS criteria: Either narration OR text, not both saying same thing

## Veto Conditions (auto-FAIL)
- High element interactivity content with no segmenting
- Novice learners presented with problems before worked examples
- Split-attention violations (diagram + separate legend)

## Scoring
- **PASS:** 6/8 or more
- **WARN:** 4-5/8 (some CLT violations present)
- **FAIL:** 3/8 or fewer

# Checklist: MEC Curso Técnico Check

**Version:** 1.0
**Agent:** mec-compliance
**When to use:** Before submitting technical course for approval or offering
**Pass threshold:** 6/7 items minimum

---

## Items

- [ ] **CNCT aligned** — Course matches a CNCT (Catálogo Nacional de Cursos Técnicos) profile | PASS criteria: Course title and content match CNCT entry exactly
- [ ] **Minimum hours met** — Meets CNCT minimum workload | PASS criteria: Total hours ≥ CNCT minimum for that course
- [ ] **Professional profile matches CNCT** — Competencies align with CNCT profile | PASS criteria: Professional profile section matches CNCT description
- [ ] **Technological axis correct** — Assigned to correct eixo tecnológico | PASS criteria: Eixo matches CNCT classification
- [ ] **CBO mapped** — CBO (Classificação Brasileira de Ocupações) codes referenced | PASS criteria: At least 1 CBO code listed and correct
- [ ] **Institutional authorization** — Institution authorized to offer technical education | PASS criteria: Institution has SISTEC registration or state authorization
- [ ] **Curriculum follows LDB** — Complies with LDB 9.394/96 requirements | PASS criteria: General education + professional education components present

## Veto Conditions (auto-FAIL)
- Course not in CNCT
- Minimum hours below CNCT threshold
- Institution not authorized for technical education

## Scoring
- **PASS:** 6/7 or more
- **WARN:** 4-5/7 (compliance issues present)
- **FAIL:** 3/7 or fewer
